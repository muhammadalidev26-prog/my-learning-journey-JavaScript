The Modern Mode

Today I'm learning about The Modern Mode: use strict in Js.

Use Strict

It enables Javscript's Strict Mode, which catches common coding mistakes and eliminate silent errors

It transforms bad syntax into explicit runtime or compile time error

Modern Javscript Syntax automatically enables strict mode so we won't need to add it ourselves but for now "use strict" is a welcome guest.

Browser Console

A browser console doesn't have use-strict mode on by default so we need to do it manually in the first line

First, Press Shift + Enter and then get on first line in say "use strict"
